## Engineering Philosophy

> 화려한 코드보다, 믿고 함께 운영할 수 있는 코드를 만드는 것을 목표로 합니다.

- 구현 전에는 **Spec · Plan · Checklist**로 요구사항을 구체화합니다.
- 구현 후에는 **테스트 · 보안 · API 계약**을 기준으로 결과를 검증합니다.
- AI 도구는 코드를 대신 쓰는 용도가 아니라, **통제 가능한 개발 도구**로 사용합니다.
- 작업 범위와 판단 근거를 문서로 남겨, **동료가 예측할 수 있게** 협업합니다.
